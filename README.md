# Парсер объявлений и цен недвижимости

Собирает **количество объявлений** и **цены публикации/продвижения** с четырёх площадок
(ЦИАН, Авито, Яндекс.Недвижимость, Домклик) по 16 городам и сводит всё в один
Excel-файл `cian_объявления.xlsx` (по листу на источник + сводный лист «Сравнение»
и «Динамика цен»).

## Что внутри

| Файл | Назначение |
|------|------------|
| `cian_parser.py` | Главный парсер (CIAN, Авито, Яндекс, Домклик + сводные листы) |
| `avito_price_api.py` | Цены Авито (публикация/выделение) через приватные API |
| `price_dynamics.py` | Перенос текущих цен в лист «Динамика цен» (Δ% по месяцам) |
| `config.py` | Все настройки: города, категории, карта URL, имена файлов |
| `run_*.bat` | Готовые ярлыки запуска под Windows |

## Установка

1. Установить **Python 3.10+** (https://www.python.org/) — при установке отметить
   «Add Python to PATH».
2. Установить зависимости:
   ```
   pip install -r requirements.txt
   ```
3. Для **Домклика** нужен установленный **Google Chrome** (используется
   undetected-chromedriver).

## Cookie и прокси (нужно подставить свои!)

В репозитории их НЕТ (это личные учётные данные). Создай из примеров:

- `cookie.txt` — cookie авторизованной сессии **CIAN** (для сбора цен). См. `cookie.txt.example`.
- `avito_cookie.txt` — cookie авторизованной сессии **Avito** (для цен Авито, живёт ~сутки).
  См. `avito_cookie.txt.example`.
- `proxies.txt` — *необязательно*, прокси для обхода бана по IP на Авито. См. `proxies.txt.example`.

Cookie берётся из браузера: DevTools (F12) → вкладка Network → любой запрос к сайту →
скопировать значение заголовка `Cookie`.

> Для сбора **количества** объявлений CIAN/Яндекс/Домклик cookie не нужен —
> он требуется только для **цен**.

## Запуск

Двойным кликом по `.bat` (Windows) или командой `python cian_parser.py <режим>`:

| Команда / .bat | Что делает |
|----------------|------------|
| `run.bat` → `python cian_parser.py cities` | CIAN — количество по 16 городам (~8 мин) |
| `run_цены.bat` → `python cian_parser.py prices` | CIAN — цены публикации (нужен `cookie.txt`) |
| `run_яндекс.bat` → `python cian_parser.py yandex` | Яндекс — количество |
| `run_авито.bat` → `python cian_parser.py avito` | Авито — количество |
| `run_домклик.bat` → `python cian_parser.py domclick` | Домклик — количество (окно Chrome ~25-30 мин, не закрывать) |
| `run_авито_цены.bat` → `python avito_price_api.py` | Авито — цены (нужен свежий `avito_cookie.txt`) |
| `run_сравнение.bat` → `python cian_parser.py compare` | Пересобрать сводный лист «Сравнение» |
| `run_обновить_всё.bat` | Полное обновление всех источников по очереди (~1-1.5 ч) |

**Перед запуском закрой `cian_объявления.xlsx` в Excel** — все парсеры пишут в него.

## Важно

- Все источники пишут в один `cian_объявления.xlsx`, поэтому запускать их нужно
  **последовательно**, а не одновременно.
- Cookie Авито протухает примерно за сутки — перед сбором цен Авито обнови его.
- Продвижение Авито собирается не всегда — при необходимости впиши вручную в лист
  «Avito Price» и запусти `run_синхр_динамика.bat`.
